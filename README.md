# 📊 Anime Data Analysis using Python

## 📌 Project Overview
This project focuses on analyzing an anime dataset using Python to extract meaningful insights. It involves data cleaning, feature engineering, and exploratory data analysis.

---

## 🛠️ Tech Stack
- **Python**
- **Pandas**
- **NumPy**
- **Jupyter Notebook**

---

## 🔍 Key Objectives
- Clean and preprocess raw dataset  
- Extract useful information from unstructured text  
- Perform data analysis to identify trends and patterns  

---

## ⚙️ Features & Implementation

### 📌 Data Cleaning
- Handled missing and inconsistent data  
- Processed text-based columns  

### 📌 Feature Engineering
- Extracted **episode count** from title  
- Calculated **airing duration (in months)**  

### 📌 Data Analysis
- Identified **top-rated anime**  
- Found **longest-running anime series**  
- Sorted and filtered dataset for insights  

---

## 📈 Key Insights
- Top-rated anime based on user scores  
- Distribution of episode counts  
- Longest-running anime by duration  

---

## 🤖 Use of AI
AI tools were used as a support system for:
- Code suggestions  
- Improving feature extraction  
- Optimizing logic  

All AI-generated outputs were reviewed and modified to ensure correctness and understanding.
